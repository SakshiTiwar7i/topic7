Activity 1

What state for each light is needed?

The state of the light lies between 0 and 1, that is, it can be greater than or equal to 0 and less than or equal to 1. Hence indicating, the light can be dimmed, switched on, switched off. (USE OF PWM and Duty Cycles)

What information do you need?

•DateTime input

•Environment luminosity readings

•User voice input

•Device input

•Switches state

•Motion sensing input

What other devices are needed?

•Smartphones, fitness bands/smart watches

•Smart switches (Manual Control)

•Voice Assisted devices (for voice recognition)

•Motion sensors (lights turn on when user enters through main door or in a specific room)

•Alarm devices (Lights turn on/off based on the time set by user)

•Microcontroller, light intensity sensor
